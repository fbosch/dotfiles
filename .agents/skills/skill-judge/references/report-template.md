# Skill Evaluation Report Template

Use this template exactly for final output.

```markdown
# Skill Evaluation Report: <skill-name>

## Summary
- Total Score: X/120 (Y%)
- Grade: <A|B|C|D|F>
- Pattern: <Mindset|Navigation|Philosophy|Process|Tool|Hybrid>
- Knowledge Ratio: E:A:R = X:Y:Z
- Verdict: <one sentence>

## Dimension Scores
| Dimension | Score | Max | Notes |
|---|---:|---:|---|
| D1: Knowledge Delta | X | 20 | ... |
| D2: Mindset + Procedures | X | 15 | ... |
| D3: Anti-Pattern Quality | X | 15 | ... |
| D4: Specification Compliance | X | 15 | ... |
| D5: Progressive Disclosure | X | 15 | ... |
| D6: Freedom Calibration | X | 15 | ... |
| D7: Pattern Recognition | X | 10 | ... |
| D8: Practical Usability | X | 15 | ... |

## Critical Issues
- <must-fix issue 1>
- <must-fix issue 2>

## Top 3 Improvements
1. <highest impact>
2. <second>
3. <third>

## Detailed Analysis
<Only include dimensions below 80% of max. For each one include:>
- What is missing or problematic
- Evidence from the skill
- Concrete fix
```

## Formatting Rules

- Keep evidence concise and specific.
- Do not include dimensions >= 80% in Detailed Analysis.
- Keep recommendations implementation-ready.
- Prioritize activation and knowledge-delta fixes first.
