# Color Tools

Vicinae extension with two small color utilities.

## Commands

- `Tailwind Colors` browses Tailwind colors, filters by search, adjusts opacity, and copies common output formats.
- `Hex Opacity Converter` converts an opacity percentage to hex alpha output.

## Usage Notes

- `Enter` and `Cmd+C` copy the primary value for the selected color or opacity result.
- Opacity values are percentages from `0` to `100`.
- Common alpha values: `100% = FF`, `75% = BF`, `50% = 80`, `25% = 40`, `0% = 00`.

## Development

pnpm install
