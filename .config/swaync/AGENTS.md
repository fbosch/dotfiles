# SwayNC Configuration - Agent Guide

SwayNC notification styling and behavior for Wayland using GTK4 CSS (not web CSS).

## Essentials

- GTK4 CSS only; avoid web CSS features like flexbox/grid/modern selectors.
- Key files: `style.css`, `config.json`.
